1000 Mile Race
author:	John Hall

**********************************************
** 	How to play 							**
**********************************************

Each player uses only the provided cards while following the Rulebook.  

7x 25 miles 		4x 75 miles 		3x targeting cheats(bounce, removal, wipe) 1 of each
6x 50 miles 		4x 100 miles 		3x library cheats(junkyard, library, shuffle) 1 of each
4x  Tires  			4x Spikes  			1x spike proof 
4x Evade  			4x Pursuit  		1x pursuit proof 
3x Token  			3x Toll  			1x toll proof 
3x Plow  			3x Roadblock 		1x roadblock proof 

**********************************************
**	How to win 								**
**********************************************

First player to have exactly 1000 miles in play wins.
